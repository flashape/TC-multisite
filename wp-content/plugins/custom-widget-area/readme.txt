======== Description ========

Have you ever wanted to show different Widgets on Pages or Posts or even inside your content?

This plugin lets you create your own widget areas, configure them by adding widgets, and replace the default sidebars or even show widgets directly in the content of Pages or Posts.

	- Individual sidebars for each Page or Post
	- 7 different styles for widgets in content area
	- Easy insert widgets in content with Shortcodes
	- Localization; English and Spanish
	- Set default sidebars for Front page
	- Set default sidebars for Home page
	- Set default sidebars for Category pages
	- Set default sidebars for Archive page
	- Set default sidebars for Pages
	- Set default sidebars for Single posts
	- Set default sidebars for 404 page (page not found)
	- Set default sidebar for Search page
	- Set default sidebar for Attachment page
	- Set default sidebar for Tag page
	- Set default sidebar for Author page
	- Set CSS for the Widgets in the content area of pages and posts


== Installation ==

1. Upload the 'custom-widget-area' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Custom Widget Area' in the left admin bar of your dashboard (under Settings)

== Frequently Asked Questions ==

Q: How can I show different widgets on Pages and Posts?

A: Simply create a custom sidebar and then go to Appearance > Widgets and configure your new sidebar. Once you have configured your sidebar with the Widgets you want to display you can apply it on any Page, Post, Category, Home (page), Frontpage, Search page, 404 page (page not found), Tag page, Author page and even to Single posts.

Q: Can I insert a widget in the content of a post or page?

A: Yes, with this plugin it is easy for you to insert a widget directly in the content. Simply click on the icon 'Insert Widget Area' in the top menu in the Visual Editor. Then you will get a pop up that shows you a drop down will all the custom widget areas that you have created. 
Please notice that you need to configure each custom widget area under Appearance > Widgets.
You can also choose between 7 different templates when you insert the widgets directly in the content of your post or page.
