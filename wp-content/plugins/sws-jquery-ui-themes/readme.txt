======== DESCRIPTION ========
This plugin is a add-on for Styles with Shortcodes. This means you need to have a valid license to Styles with Shortcodes in order to use this plugin. It will add 8 additional jQuery UI themes to your SWS plugin giving you even more styling options for the built in Accordions, Dialog boxes, Icons, Tabs and Toggles.

New themes added:
* Cupertino
* Dark-hive
* Eggplant
* Pepper-grinder
* Redmond
* Sunny
* UI-darkness
* UI-lightness
* Flick



== INSTALLATION ==

1. Upload the 'sws-jquery-ui-themes' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

== FREQUENTLY ASKED QUESTIONS ==

Q: After activating the plugin how do I use the Shortcodes SWS jQuery UI Themes?
A: In order to use the SWS jQuery UI Themes plugin you must have a valid license of Styles with Shortcodes for WordPress. 

Go to a Post or Page and click on the "S" icon above the visual RTF editor. Then you will get a pop up. Choose the category "Tooltips" from the drop down and then choose the effect you would like to insert. Please notice that the jQuery UI themes provide additional styling options for the following Shortcodes: Accordions, Dialog Boxes, Icons, Tabs and Toggles.

Q: Can I add my own jQuery themes to the plugin?
A: Yes, you can. Just go to http://jqueryui.com/themeroller/ and then download additional Themes for free. You can even create your own themes and download them. Once you have downloaded them simply upload them to the themes folder

/wp-content/plugins/sws-jquery-ui-themes/themes/

Q: What happens if I customize some of the Shortcodes in Styles with Shortcodes and they don't work any more?
A: If you mess up the Shortcode you can always go to the Options Panel and then click on "Bundles". There you will find a drop-down that contains the bundles you have installed. Choose the bundle you want the restore and then simply "Click to restore bundle". This way you restore everything and it should work again.
