uBillboard Help

uBillboard is a Premium Slider Plugin for WordPress by uDesignStudios.

This page lists all available billboards that you have set up on your site. In the Shortcode column, you can see the shortcode you can use to put the billboard on a page or in a post.

If you hover over a line, you can see additional actions that you are able to perform with the billboard.

  • Edit
  • Delete
  • Preview
  • Export

You can also bulk delete, or export billboards using the bulk options.
